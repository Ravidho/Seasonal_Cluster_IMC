# Seasonal_Cluster_IMC
Dominant seasonal rainfall patterns across the Indonesian Maritime Continent (IMC) and its surrounding oceans (90°E–145°E, 11°S–10°N) using IMERG V07B satellite precipitation data (0.1°-monthly) and rain gauge observations from 2001 to 2023.
